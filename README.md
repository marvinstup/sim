문서 1. 모델 비교 보고서

같은 입력을 네이토 모델 드롭다운에서 바꿔가며 비교 (예: GPT-5 / Claude Opus 4.8 / Gemini 3.1 Pro)
네이토에서 기록

순서: 모델 A / 모델 B / 모델 C
1. 정확성(사실·수치): 4 / 5 / 3  → C는 '확인 필요' 항목 일부 누락
2. 한국어 자연스러움: 4 / 4 / 5
3. 형식 준수: 3 / 5 / 4  → B가 불릿 구조 가장 정확
4. 긴 문맥 유지: 4 / 5 / 3
5. 속도·안정성: 5 / 4 / 4
합계 → 20 / 23 / 19  →  최종 선정: 모델 B

선정이유
1) 대부분은 이제 고급모델이라 환각이 나오는 경우가 드물었음
2) 형식준수에서 가장 뛰어났음
3) c모델의 경우 일부 누락이 있음
